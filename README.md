# SiteBot
SiteBot, işletmelerin kendi sunucusunda çalışan ücretsiz, Docker destekli AI altyapısıdır. Web sitesi, CRM, admin panel, WhatsApp, Telegram ve e-posta botlarını oluşturur. Kullanıcı şirketini anlatır, modülleri seçer; AI taslak hazırlar. Kullanıcı konuşarak düzenler, önizler, onaylar ve güvenle yayınlar. Veriler tamamen kendi sunucusunda kalır vb.
